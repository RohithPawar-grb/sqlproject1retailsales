Retail Sales Analysis – SQL Project

This is a beginner-friendly SQL project where I performed data cleaning, exploration, and simple analysis on a retail sales dataset.

Project Goals

• Create a retail sales database
• Clean the data by removing NULL values
• Explore the dataset
• Answer basic business questions using SQL queries

Dataset Columns

transactions_id – Unique ID for each transaction
sale_date – Date of sale
sale_time – Time of sale
customer_id – Customer ID
gender – Customer gender
age – Age of customer
category – Product category
quantiy – Quantity sold
price_per_unit – Price of one unit
cogs – Cost of goods sold
total_sale – Total sale amount

Data Cleaning Steps

• Checked for NULL values
• Removed rows with missing important fields
• Verified total row count after cleaning

Analysis Performed

• Total number of sales
• Unique customers
• Category-wise sales
• Customers’ average age (Beauty category)
• Transactions above 1000 sale amount
• Gender-wise transactions
• Best selling month using window functions
• Top 5 customers
• Sales based on time shifts (Morning, Afternoon, Evening)

How to Run the Project

1. Create the database
2. Import the retail dataset
3. Run the SQL cleaning queries
4. Run the analysis queries to get insights

Conclusion

This project helped me understand SQL basics such as data cleaning, grouping, filtering, date functions, and window functions.
